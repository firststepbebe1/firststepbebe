
First Step Bebe – FINAL (Brevo SMTP + CMS + Admin + Cart)

1) Urcă proiectul pe Netlify.
2) Setează Environment Variables:
   - BREVO_SMTP_HOST = smtp-relay.brevo.com
   - BREVO_SMTP_PORT = 587
   - BREVO_SMTP_USER = firststepbebe1@gmail.com
   - BREVO_SMTP_PASS = <SMTP key din Brevo>
   - ORDER_EMAIL_TO = firststepbebe1@gmail.com
   - ORDER_EMAIL_FROM = First Step Bebe <firststepbebe1@gmail.com>
3) (Opțional, pentru CMS live): conectează site-ul la GitHub, apoi Identity + Git Gateway + Invite user, mergi la /cms/.
4) Editează produse & poze din CMS sau admin.html.
